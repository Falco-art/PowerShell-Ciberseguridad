#PowerShell-Ciberseguridad

Este repositorio ha sido creado para la publicación, documentación y colaboración de proyectos desarrollados en PowerShell, integrando herramientas profesionales de control de versiones

¿Qué contiene?
Una colección de scripts automatizados diseñados para fortalecer la postura de seguridad en entornos Windows, organizados por carpetas temáticas

¿Qué scripts se incluyen?
Ejercicio 1 (Verificador de Integridad) un script que calcula hashes SHA256 y MD5 para asegurar que los archivos no hayan sido alterados
Ejercicio 2 (Auditoría de Privilegios) una herramienta para listar los miembros del grupo de administradores locales y detectar posibles riesgos de acceso.

¿Qué tareas de ciberseguridad resuelven?
Prevención de ejecución de archivos maliciosos mediante la validación de firmas digitales y hashes
Gestión de la superficie de ataque mediante el control estricto de los privilegios de usuario en el sistema

¿Qué aprendí al desarrollarlos?
Aprendí a utilizar GitHub como una plataforma profesional para el mantenimiento de código, además de reforzar mis competencias en documentación técnica y control de versiones mediante el uso de GitHub Desktop

Durante el desarrollo de esta evidencia, se fortalecieron las competencias en el uso de herramientas de control de versiones como GitHub Desktop y la implementación de buenas prácticas en la publicación de scripts de seguridad, se logró una integración fluida entre el código local y el repositorio remoto, asegurando la trazabilidad de cada cambio realizado
